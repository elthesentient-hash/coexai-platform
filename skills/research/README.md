# Research Skills Library for AI Agents

Complete research toolkit for CoExAI agents - web search, academic papers, data analysis, and more.

## Skills Overview

| Skill | Description | Cost |
|-------|-------------|------|
| web-deep-research | Multi-source web research with synthesis | Free |
| academic-research | Scholar, arXiv, paper analysis | Free |
| data-analysis | CSV, JSON, dataset analysis | Free |
| trend-analysis | Google Trends, social trends | Free |
| competitor-analysis | Market research, pricing intel | Free |
| patent-research | USPTO, patent analysis | Free |
| news-monitoring | Real-time news tracking | Free |
| reddit-research | Subreddit analysis, sentiment | Free |

## Quick Start

```bash
# Web research
./skills/research/web-research.sh "AI trading bots 2025"

# Academic papers
node skills/research/academic-research.js "machine learning finance"

# Data analysis
node skills/research/data-analysis.js dataset.csv

# Trends
./skills/research/trends.sh "crypto trading"
```

## Integration

Add to agent's SKILL.md:

```yaml
skills:
  - youtube-research
  - web-deep-research
  - academic-research
  - data-analysis
  - trend-analysis
```
